# Experiment Results

This directory contains the reference artifacts generated by the documented default configurations.

## Cross-Experiment Summary

Run the complete core suite with:

```bash
vision-playground run all
```

This command runs five registered experiments, performs 165 evaluations, and writes [experiment_summary.csv](experiment_summary.csv).

| Experiment | Evaluations | Summary evidence |
| --- | ---: | --- |
| `thresholding` | 12 | Adaptive thresholding reaches 0.847 IoU under uneven illumination |
| `adaptive-sensitivity` | 120 | The best tested shared configuration averages 0.830 IoU |
| `denoising` | 6 | Median filtering reaches 0.992 IoU under salt-and-pepper noise |
| `edge-detection` | 9 | Median filtering reaches 1.000 edge F1 under salt-and-pepper noise |
| `labeled-dataset` | 18 | Fixed-inset GrabCut averages 0.745 IoU on the selected six-image subset |

The reference values are intentionally scoped to specific questions and conditions. They are not a leaderboard: different rows use different targets, metrics, and aggregation policies.

## Summary Schema

| Column | Meaning |
| --- | --- |
| `experiment` | Stable CLI identifier |
| `question` | Research question addressed by the experiment |
| `evaluations` | Number of method-condition evaluations |
| `summary_metric` | Metric used for the selected evidence |
| `reference_value` | Reproduced value for that evidence |
| `evidence_scope` | Exact method and condition represented |
| `metrics_path` | Detailed metrics CSV, relative to this directory |
| `visual_artifact` | Visual comparison, relative to this directory |

The detailed CSV and image for each row remain the source of truth. The summary is an index for review, not a substitute for experiment-specific interpretation and limitations.
